## 🚀 **Como Usar a Integração**

### **1. Instalar Dependências:**
```bash
cd frontend
npm install
```

### **2. Iniciar Backend:**
```bash
cd backend
npm start
```

### **3. Iniciar Frontend:**
```bash
cd frontend
npm run dev
```

### **4. Acessar Funcionalidades:**

#### **📋 Lista de Viveiros (Backend):**
```
http://localhost:5173/viveiros-backend
```
- Lista todos os viveiros do backend PostgreSQL
- Criar novos viveiros
- Editar viveiros existentes
- Deletar viveiros

#### **📝 Edição de Viveiro (Backend):**
```
http://localhost:5173/viveiro/:id
```
- Carrega dados do backend
- Formulário com dados reais
- Salva alterações no backend
- Interface profissional

#### **📊 Dashboard (Backend):**
```
http://localhost:5173/viveiro/:id/dashboard
```
- Dados sincronizados com backend
- KPIs em tempo real
- Gráficos com dados reais
